## Project Summary

**Visit Processor Testing** is an automated testing framework for the Visit Document Processor (VDP) with AI-powered text generation capabilities.

### Purpose
- Automated testing for VDP functionality
- AI-powered test data generation
- Validate VDP workflows and outputs

### Target Users
- Developers working on VDP
- QA testers
- CI/CD pipelines

### Technology Stack
- Python
- AI text generation

### Related Projects
- VDP (Visit Document Processor) - the main application being tested
- excel-vba-projects - legacy VDP codebase

---

## Project Overview

This testing framework provides automated validation for the Visit Document Processor. It uses AI to generate realistic test data for probation visit scenarios.
